# User Stories

## 역할 정의

| 역할 | 설명 | 인증 타입 |
|------|------|---------|
| **구직자** | 외국인 구직자, 이력서 작성 및 채용공고 지원 | Google OAuth / 이메일 |
| **기업 담당자** | 채용 담당자, 공고 등록 및 지원자 관리 | 이메일 (company) |
| **관리자** | 플랫폼 운영 관리자 | 별도 인증 |

---

## 구직자 Stories

### US-001: 회원가입
**As a** 구직자
**I want to** Google 계정으로 간편 가입하거나 이메일로 가입하고 싶다
**So that** 빠르게 플랫폼을 이용할 수 있다

**AC:**
- Google OAuth 로그인 가능
- 이메일 인증 코드 발송 및 검증
- 가입 완료 후 메인 페이지로 이동

---

### US-002: 채용공고 탐색
**As a** 구직자
**I want to** 조건에 맞는 채용공고를 찾고 싶다
**So that** 원하는 직무/지역/급여의 일자리를 발견할 수 있다

**AC:**
- 고용형태 필터 (정규직/계약직/파트타임 등)
- 근무지 검색
- 급여 정렬
- 공고 제목 검색
- 결과 없을 때 EmptyState 표시

---

### US-003: 이력서 작성
**As a** 구직자
**I want to** 온라인으로 이력서를 작성하고 싶다
**So that** 채용공고에 바로 지원할 수 있다

**AC:**
- 기본 정보, 학력, 경력, 어학, 자격증 섹션
- 여러 이력서 관리 (복수 이력서)
- 임시 저장 (페이지 이탈 경고)
- 이력서 삭제

---

### US-004: 채용공고 지원 (P0 — 서버 미구현)
**As a** 구직자
**I want to** 채용공고에 내 이력서를 첨부해서 지원하고 싶다
**So that** 기업 담당자에게 내 정보를 전달할 수 있다

**AC:**
- 이력서 선택 모달
- 지원 완료 확인
- 중복 지원 방지
- 지원 내역에서 상태 확인
- ⚠️ `POST /api/applications` 서버 구현 필요

---

### US-005: 직무 적합도 진단
**As a** 구직자
**I want to** 내 적성에 맞는 직무를 알고 싶다
**So that** 더 적합한 공고에 집중할 수 있다

**AC:**
- 진단 질문 응답 (다음/이전 네비게이션)
- 결과 저장 및 재조회
- 결과 기반 공고 추천 (P2)

---

### US-006: 프로필 관리
**As a** 구직자
**I want to** 내 기본 정보와 연락처를 관리하고 싶다
**So that** 최신 정보로 채용 담당자와 소통할 수 있다

**AC:**
- 이름, 생년월일, 프로필 사진 수정 (`PATCH /api/me`)
- 이메일, 전화번호 수정 (`PATCH /api/contact`)
- 알림 설정 (`PATCH /api/account-config`)

---

## 기업 담당자 Stories

### US-101: 채용공고 등록
**As a** 기업 담당자
**I want to** 채용공고를 등록하고 싶다
**So that** 적합한 외국인 인재를 모집할 수 있다

**AC:**
- 제목, 직무 내용, 근무지, 고용형태, 급여, 기간 입력
- 공고 수정/삭제
- 공고 목록에서 내 회사 공고 관리

---

### US-102: 지원자 관리 (P1 — 서버 미구현)
**As a** 기업 담당자
**I want to** 공고에 지원한 사람들을 확인하고 싶다
**So that** 효율적으로 채용 절차를 진행할 수 있다

**AC:**
- 공고별 지원자 목록
- 지원자 이력서 조회
- 지원자 상태 업데이트 (검토 중/합격/불합격)
- ⚠️ `GET /api/posts/company/{id}/applicants` 서버 구현 필요

---

### US-103: 기업 프로필 관리
**As a** 기업 담당자
**I want to** 회사 소개를 업데이트하고 싶다
**So that** 구직자가 우리 회사를 신뢰할 수 있다

**AC:**
- 회사명, 업종, 규모, 소개 수정 (`PATCH /api/company-profile`)
- 사업자등록번호 검증 (국세청 API 연동)

---

## 관리자 Stories

### US-201: 사용자 관리
**As a** 관리자
**I want to** 전체 사용자 목록을 조회하고 관리하고 싶다
**So that** 플랫폼 신뢰도를 유지할 수 있다

**AC:**
- 구직자/기업 목록 조회
- 계정 활성화/비활성화
- 부적절한 콘텐츠 신고 처리

---

### US-202: 공고 모더레이션
**As a** 관리자
**I want to** 등록된 채용공고를 검토하고 싶다
**So that** 허위/불법 공고를 차단할 수 있다

**AC:**
- 전체 공고 목록 조회
- 공고 삭제/숨김 처리
- 공고 신고 처리

---
